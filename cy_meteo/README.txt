Pour compiler, vous devez aller dans le terminal du dossier cy_meteo
et lancer la commande make. Le projet est ainsi compiler. Pour pouvoir executer le projet, 
vous devez lancer la commande ./main -f <nom du ficher d'entree> -o <nom
su fichier de sortie> 

Vous pouvez ensuite rajouter a la commande de base : -r pour un trie decroissant 
et --tab/--abr/--avl pour choisir votre methode de triage.
